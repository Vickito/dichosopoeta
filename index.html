Blog o Página Web Personal

Disponer de un blog o una página web personal es imprescindible para cualquier autor. Sirven para promocionarse y acercarnos a nuestros posibles lectores.

Una página web personal requiere ciertos conocimientos técnicos, y conlleva algunos costes. Trabajaremos sobre la base de Blogs gratuitos, que es la forma más fácil de crear y actualizar una página Web...

¿Qué es un Blog?

Básicamente, un Blog es un espacio personal de escritura en Internet donde el autor publica artículos o noticias. Estos pueden contener texto, imágenes e hipervínculos...

Elementos de un Blog

En la portada del weblog aparecen primero las anotaciones más recientes. Cada uno de estos post incluye un título, la fecha de publicación, el nombre del autor, y un enlace que lleva a un formulario donde los visitantes pueden escribir sus comentarios...

¿Por dónde Empezar?

Para conseguir un espacio en Internet y comenzar a publicar tus noticias, artículos y reflexiones existen en la actualidad muchas opciones...
